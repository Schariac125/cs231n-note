## 1.线性代数的本质

**到底什么是线性无关和相关**：如果现在有两个向量，这两个向量共线，那么他们数乘特定的值之后相加得到的依然是他们那么维度的向量，但如果不共线呢？那么他们相加就可以得到更高维度的向量，例如线变成面这一类的。也就是说，如果说几个向量都为他们数乘特定值之后相加升维做出了贡献，那么就说他们是线性无关的，那其实这样子想的话，最大线性无关组的意义也就很明了了

**什么又是线性变化**：线性变化有两个非常重要的点，拿二维的平面直角坐标系来举例，经过线性变换之后原点不可以发生改变，原本在这个坐标系上两点连成的任何一条线都不能变成弯曲的，可以理解为这就是一种保持平行且等距的变换

**矩阵的本质是什么**：事实上，我们可以把矩阵看作是由几个向量组成的组合。而矩阵则代表着如何操纵空间，可以理解为它是对基向量的变换。

**矩阵的乘积可以怎么理解**：如果把矩阵的乘法看作是一个线性变换的操作，假设乘上一个矩阵就是一次线性变换，那么如果我们要进行多次线性变换，就需要乘上多个特定变换的矩阵。但事实上，这些矩阵理论上讲都可以复合成一个矩阵，即使用一个矩阵就可以完成所有操作，这个复合矩阵就是前面所有矩阵的总影响。两个矩阵相乘就是两种线性变换相继作用，但实际上可以合为一种

## 2.图像分类

图像的原始表达形式是一个表示亮度值的三维数组，而图像分类器就是基于这个三维数组来操作的

> 一个好的图像分类模型必须对这些变化的所有交叉乘积保持不变，同时保持对类间差异的敏感性。

事实上，我们直接去编写一个识别特定物体的算法并不现实，更现实的方法是把计算机当作一个学生然后通过告诉它这张图是什么，来教会它学习。而这种算法会查看这些实例并且分析他们的特征最后学会。这种依赖于带标签图像的训练数据集的方法被称为数据驱动方法。毕竟这世上物体太多，要每一个都去针对性的编写算法去识别这很抽象而且局限性很强。

这个过程是这样的

- 输入：输入包含了一组N张图像，每一张图像都被标记为K个不同类别之一，这就是训练集
- 学习：模型分析并理解图像特征
- 评估：最后拿一组不存在于数据集中的新图像来评估

### 最近邻分析器（NN）

最近邻分析器其实是一个不是很常用的东西，但是可以帮助我们理解图像分类问题的基本方法就是了

如果要比对几张图像最直观的想法其实是去一个一个比对像素，然后求出综合，一个比较合理的选择是采用曼哈顿距离
$$
d_1 (I_1, I_2) = \sum_{p} \left| I^p_1 - I^p_2 \right|
$$
下面是cs231n的笔记示例代码

```python
import numpy as np
from data_utils import load_CIFAR10
from tqdm import tqdm

Xtr, Ytr, Xte, Yte = load_CIFAR10("data/cifar-10-batches-py")
Xtr_rows = Xtr.reshape(Xtr.shape[0], 32 * 32 * 3)
Xte_rows = Xte.reshape(Xte.shape[0], 32 * 32 * 3)


class nn:
    def __init__(self):
        pass

    def train(self, X, y):
        self.Xtr = X
        self.ytr = y

    def predict(self, X):
        num_test = X.shape[0]
        Ypred = np.zeros(num_test, dtype=self.ytr.dtype)

        for i in tqdm(range(num_test), desc="正在分类图片"):
            distance = np.sum(np.abs(self.Xtr - X[i,:]), axis = 1)
            min_index = np.argmin(distance)
            Ypred[i] = self.ytr[min_index]

            
        return Ypred


if __name__ == "__main__":
    n = nn()
    n.train(Xtr_rows, Ytr)
    print("开始预测...")
    Yte_predict = n.predict(Xte_rows[:100, :])
    accuracy = np.mean(Yte_predict == Yte[:100])
    print("accuracy: %f" % (np.mean(Yte_predict == Yte[:100])))
```

这个代码采用了L1距离作计算，在只有一百张图像的情况下准确率可以达到34%（其实是因为样本数量太少了

同样的，还有采用L2距离进行计算的方法
$$
d_2 (I_1, I_2) = \sqrt{\sum_{p} \left( I^p_1 - I^p_2 \right)^2}
$$
只需要修改一行代码`distances = np.sqrt(np.sum(np.square(self.Xtr - X[i,:]), axis = 1))`

### k-最近邻分类器（k-NN）

knn的思路非常简单，在nn的基础上加入一个k参数，而这个k参数代表着什么呢？

事实上，这个k参数加入之后就代表着我们不再只是找训练集里面最接近的单个图像，而是找出k个接近的图像，然后对他们进行投票。而很明显的就是，所谓的NN其实就是k等于1的knn

**事实上，k值越大分类器越能够抗干扰的影响，一般而言k为3/5是比较合适的，k如果是1就没有意义了，而k太大就会导致模型犹豫不决，造成过拟合**

但这么说其实非常笼统，那么k到底应该设置为多少？

先讲一个题外话，我们发现在分类器里面有许多不同的距离函数可以供选择，例如说L1/L2，还有很多我们没有考虑过的，这些选择都被称之为超参数。通常来讲，参数和超参数的选择和设置并不是一件显而易见的事情。

**“？这准确率是不是有点”**

那是因为你过拟合了……现实生产里面的数据可不是像数据集一样的。事实上是，在最后评估之前都不要去碰测试集
